# rentalcar-app
A rental car application based on Spring Boot

This application is designed based on Spring MVC architecture and is using following technologies:

Frontend:
  - HTML, CSS
  - Bootstrap 4
  - Thymeleaf
  
Backend:
  - Spring Boot
  - Spring Security
  - Spring JDBC
  - Spring Data JPA
  - Hibernate
  - MySQL
  - Maven
